A Hospital Patient Management System built in C++ using nested structures, pointers, and dynamic memory allocation.
# 🏥 Hospital Patient Management System

## Overview
A C++ console application to manage hospital patient records.

## Features
- Store N patient records dynamically
- Nested doctor information inside patient record
- Search patient by ID
- Update doctor details
- Display updated patient record

## Concepts Used
- Nested Structures
- Pointer to Structure Array
- Dynamic Memory Allocation
- Reference Return from Function
- Search Algorithm

## How to Run
- Clone the repository
- Open in any C++ compiler
- Run main.cpp
